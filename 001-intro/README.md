## Какие файлы в папке config
Там как я поняла конфигурация приложения, причем считывает чувствительные данные из .env
Видимо чтобы в гит или подобные видимые места без опаски пушить файлы, а сами значения туда не попадали
Ну и можно создавать свои конфиги и потом их значения считывать через функцию config

## Где основная бизнес-логика
В папке app, вся структура там. Ещё может в resources/views, там хранятся html, и js из которого собирается бандл если бизнес-логика
может хранится в фронт-энде.
