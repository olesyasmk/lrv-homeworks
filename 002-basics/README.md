# Ответы на вопросы

## Для чего нужны миграции? Какую роль они играют или какую проблему решают?
Для создания схем БД. Так как структура БД создается один раз. Ещё версионирование, колонки могут добавляться и удаляться
и миграции вместе с гит позволяют смотерть историю и помогают в проектах где несколько разработчиков

## Для чего нужен Eloquent ORM?
Чтобы иметь объектные представления записей из БД и чтобы в случае смены БД не переписывать SQL под специфику СУБД
